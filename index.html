BlackArch WSL

For installation details, please refer to the wiki

1. Download the BlackArch.7z file
2. Extract all files to same directory
(Please extract Blackarch.zip to a folder. Avoid using Program Files)
3. Run BlackArch.exe to Extract rootfs and Register to WSL

(The “.exe” filename is being used to register the WSL instance name.**If want to have have multiple installs all you have to do is rename it, so that you can register with a different name. )
